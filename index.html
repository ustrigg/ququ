ASR 服务器设置

Paraformer: Docker 部署，CPU/GPU 均可，启动快速。
Qwen3-ASR: Docker 部署，需要 GPU (~8GB显存)，精度更高。
自动选择: 优先 Qwen3，失败自动回退到 Paraformer。
启动命令: python qwen_asr_server.py start

翻译设置

翻译功能使用本地部署的 GPT-OSS-20B 模型，将中文口语优化翻译为英文书面语。按 Ctrl+Shift+M 可快速切换转写/翻译/双语模式。

文本优化设置

文本优化使用 LLM 将口语化的语音转写结果优化为自然的书面语。可在主窗口中通过开关按钮启用/禁用。提示词末尾会自动追加「原文：{转写文本}\n优化后：」。

录音反馈设置

按 Ctrl+Shift+R 开始/停止录音，按 Ctrl+Shift+M 切换模式。录音完成后会自动进行语音识别并根据设置处理结果。
