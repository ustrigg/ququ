# 快捷键修改说明

## 修改内容

由于 F4 快捷键已被系统占用，我们已将快捷键修改为：

### 新快捷键

- **[ 键** (左方括号): 开始/停止录音
- **] 键** (右方括号): 切换工作模式

### 工作模式切换顺序

按 `]` 键循环切换：
1. 转写模式 →
2. 翻译模式 →
3. 双语模式 →
4. 返回转写模式

## 技术说明

### Electron globalShortcut 限制

Electron 的 `globalShortcut.register()` 不支持组合键（如 `Tab+\`），只支持：
- 单个键：`A`, `[`, `]`, `F1-F12` 等
- 修饰键组合：`Ctrl+A`, `Shift+B`, `Alt+C`, `Command+D` (Mac)

### 为什么选择 [ 和 ]

1. **位置相邻**：两个键紧挨着，方便操作
2. **不常用**：不会与其他应用冲突
3. **易记**：左括号开始/停止，右括号切换模式
4. **单手操作**：可以用一只手完成所有操作

### 快捷键冲突检测

快捷键注册时会在控制台输出日志：
```
Record hotkey registered: [ success: true
Mode toggle hotkey registered: ] success: true
```

如果 `success: false`，说明快捷键已被其他应用占用。

## 配置文件

快捷键配置保存在 `config.json`：

```json
{
  "hotkey": "[",
  "modeToggleHotkey": "]",
  ...
}
```

可以手动编辑此文件来修改快捷键。

## 常见问题

**Q: 快捷键不生效怎么办？**
A:
1. 检查控制台日志，确认注册成功
2. 尝试更换其他快捷键
3. 确保没有其他应用占用该快捷键
4. 重启应用

**Q: 如何恢复默认快捷键？**
A: 删除 `config.json` 文件，重启应用即可。

**Q: 可以使用组合键吗？**
A: 可以，但只支持标准组合键，如：
- `Ctrl+[`
- `Alt+]`
- `Shift+F8`
- 不支持：`Tab+\`, `[+]` 等非标准组合

**Q: Mac 上的快捷键是什么？**
A: 相同，[ 和 ]，或者使用 `Command+[` 和 `Command+]`
