## 如何使用

### 1. 基本录音流程

1. 按 **F10** 开始录音
   - 系统托盘图标显示 "Ququ Voice Input - Recording"
   - 如果打开设置界面，会看到 "正在录音..."

2. 说出您要转写/翻译的内容

3. 按 **F10** 停止录音
   - 自动进行语音识别
   - 根据当前模式处理结果
   - 结果自动复制到剪贴板

### 2. 切换工作模式

按 **F11** 循环切换：

**转写模式** (默认)
- 仅识别中文，直接输出
- 适合：中文输入、笔记

**翻译模式**
- 中文 → 英文
- GPT-OSS-20B 优化为书面语
- 适合：英文邮件、文档

**双语模式**
- 同时显示中英文对照
- 格式：
  ```
  中文: 你好世界

  English: Hello World
  ```

### 3. 查看当前模式

- 方法1: 右键系统托盘图标 → "Show Settings"
- 方法2: 查看设置界面的蓝色状态栏

## 翻译功能设置

### 默认配置

```json
{
  "translationEnabled": true,
  "translationServerUrl": "http://192.168.2.2:1234",
  "translationModel": "gpt-oss-20b",
  "translationStyle": "professional"
}
```

### 翻译风格

在设置界面可选择：

- **专业/正式** (professional) - 适合商务邮件、正式文档
- **日常/轻松** (casual) - 适合日常对话、社交媒体
- **学术/严谨** (academic) - 适合学术论文、研究报告
- **商务** (business) - 适合商务谈判、合作沟通
- **技术文档** (technical) - 适合技术文档、开发文档

## 常见问题

**Q: 如何知道应用是否在运行？**
A: 检查系统托盘（右下角）是否有麦克风图标

**Q: 快捷键不响应？**
A:
1. 确认启动日志显示 `success: true`
2. 重启应用
3. 尝试更换快捷键（在设置界面修改）

**Q: 翻译失败？**
A:
1. 确认翻译服务器运行中 (http://192.168.2.2:1234)
2. 检查网络连接
3. 查看控制台错误日志
4. 会自动回退到原始中文文本

**Q: 如何自定义快捷键？**
A:
1. 打开设置界面
2. 修改"录音热键"和"模式切换热键"
3. 保存设置
4. 快捷键会立即生效
